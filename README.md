# ASCII-Art
## Deploy
```bash
git clone https://github.com/obe1one/ASCII-Art.git
cd ASCII-Art
git checkout deploy
###############################
npm install    || yarn
npm run server || yarn server
npm run start  || yarn start
```
* Default URL: http://140.112.30.33:3000
